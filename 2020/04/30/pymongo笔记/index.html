前言

依靠pymongo包来对mongoDB数据库进行操作

一、pymongo安装

首先通过pip list或者通过import pymongo，查看自己装没装pymongo。

出错提示版本匹配有问题。

通过Anaconda中安装pymongo环境，失败。

重新更新pip版本

更新成功后，再pip install pymongo，成功！！
